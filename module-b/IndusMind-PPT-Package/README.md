# IndusMind PPT 资料包

本文件夹汇总了制作、修改和讲解 IndusMind 项目 PPT 所需的演示文稿、说明文档与视觉素材。

## 目录

- `PPT/`
  - `IndusMind-content-deck-slideforge-images.pptx`：推荐使用的图片增强版，21 页。
  - `IndusMind-content-deck-slideforge.pptx`：纯结构化图形版，16 页。
- `DOCS/`
  - `project-introduction-for-ppt.md`：PPT 主体内容、页面提纲和讲解要点。
  - `architecture.md`：系统架构与四层诊断流程。
  - `knowledge-rag.md`：混合 RAG 与知识库设计。
  - `remote-api.md`：异步诊断 API 与远程调用方式。
  - `frontend-handoff.md`：前端联调和结果展示说明。
  - `README.md`：项目文档索引。
- `IMAGES/`
  - 5 张 PNG：适合直接插入 PowerPoint。
  - 5 张 SVG：适合无损缩放和二次编辑。
  - `README.md`：各图片的用途说明。

## 推荐使用方式

优先打开图片增强版 PPT，再从 `DOCS/project-introduction-for-ppt.md` 补充演讲稿或页面内容。需要调整视觉时，优先使用 `IMAGES/` 中的 SVG；只需快速插图时使用 PNG。

本资料包不包含 `.env`、API Key 或其他私密配置。
