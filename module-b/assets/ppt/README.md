# IndusMind PPT 图片素材

所有素材均为 16:9（1600×900），同时提供：

- `.png`：直接插入 PowerPoint；
- `.svg`：无损缩放，可在支持 SVG 的软件中修改颜色或元素。

| 文件 | 建议用途 |
|---|---|
| `01-indusmind-cover` | 封面、章节页、结束页背景 |
| `02-four-layer-architecture` | 四层架构与完整诊断流程 |
| `03-hybrid-rag` | 本地结构化知识 + Dify 语义知识 |
| `04-cloud-edge-deployment` | 云端前端、ECS/frp、本地 Agent 部署 |
| `05-human-review-gate` | 置信度门禁与人工复核 |

图片统一使用深蓝、工业灰、青色和少量橙色，与
`docs/project-introduction-for-ppt.md` 中的视觉建议一致。

图内使用英文标签以避免不同 PowerPoint 环境出现中文字体替换或乱码；可在 PPT 中覆盖中文标题和注释。
